## Agent skills

### Issue tracker

작업 이슈는 `.scratch/<feature>/` 아래의 로컬 Markdown 문서로 관리합니다. 외부 PR은 triage 대상이 아닙니다. 자세한 내용은 `docs/agents/issue-tracker.md`를 참고합니다.

### Triage labels

기본 triage 상태인 `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`를 사용합니다. 자세한 내용은 `docs/agents/triage-labels.md`를 참고합니다.

### Domain docs

단일 컨텍스트 구조로 루트 `CONTEXT.md`와 `docs/adr/`를 사용합니다. 자세한 내용은 `docs/agents/domain.md`를 참고합니다.
